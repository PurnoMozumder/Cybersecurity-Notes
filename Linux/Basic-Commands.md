# Linux Basic Commands

## 📂 Navigation

| Command | Purpose |
|---------|---------|
| `pwd` | Show current directory |
| `ls` | List files |
| `cd` | Change directory |

## 📄 File Management

| Command | Purpose |
|---------|---------|
| `touch` | Create file |
| `cat` | Read file |
| `mkdir` | Create folder |

mkdir Cybersecurity
cd Cybersecurity
touch notes.txt
